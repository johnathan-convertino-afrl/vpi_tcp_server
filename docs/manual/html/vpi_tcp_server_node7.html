Building

The all VPI TCP Server source files are written in C to target the VPI API from Verilog 2001. They should simulate in any modern simulation tool that has VPI support. The library comes as a fusesoc packaged core and can be included in any other testbench. Be sure to make sure you have meet the dependencies listed in the previous section.
